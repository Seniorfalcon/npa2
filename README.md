# npa2

![npa2 screenshot](https://github.com/leifarriens/npa2/blob/master/screen.png?raw=true)

Dependency overview tool for javascript projects.

Project directory needs to contain `package.json` file.

## Installation

```sh
npm i -g npa2
```

## Usage

```sh
npa2
```

### CLI flags

Usage:

```sh
npa2 [...flags]
```

#### `--version, --V`

Output the version number.

#### `--help, -h`

Display help for command.

#### `--dev, -D`

Get devDependencies overview.
